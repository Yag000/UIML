Usage instructions. We support four logics:
  • intuitionistic Strong Löb logic (iSL),
  • intuitionistic logic (IL) (use the option iSL and give an input with no □),
  • Gödel-Löb logic (GL), and
  • basic modal logic (K).
You may use the following symbols:
→ -> implication
¬ ~ negation
∧ & conjunction (and)
∨ | disjunction (or)
□ [] necessity (box)
⋄ <> possibility (diamond)
⊤ T verum
x0 propositional variable to be quantified
x1, x2, ... other propositional variables

The quantifiers are always with respect to the variable x0.
